Formularios HTML

Son elementos que permiten el intercambio de datos entre cliente y servidor.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle al menos 2 atributos action="..." el cual indica que debe suceder al enviar el formulario; y method="..." que indica como se debe realizar el intercambio de los datos.

  • GET: metodo cuya prioridad radica en la respuesta del servidor, ya que los datos se envian en texto plano y permanecen visibles en la barra de direcciones. Se utiliza principalmente en Buscadores.
  • POST: metodo cuya prioridad esta en proteger los datos enviados por el usuario, ya que cifra los mismos antes de realizar el envio. Se utiliza en formularios de registro, acceso, contacto, etc...

¿Como estan compuestos?

Dentro del formulario podemos colocar cualquier etiqueta, pero necesariamente vamos a utilizar etiquetas LABEL que funcionan a modo de rotulo, para que el usuario sepa como debe completar el mismo; y los Campos (INPUT/ TEXTAREA / SELECT) que permiten al usuario seleccionar o completar con sus datos.

  • name="..." corresponde al nombre con el que indeficara el campo, este puede ser compartido en caso de selecciones de opcion.
  • type="..." corresponde al tipo de dato admitido por el campo de formulario, en caso de los input es obligatorio.
  • value="..." corresponde al valor que sera enviado al utilizar el formulario y es obligatorio en caso que el usuario seleccione.

Tablas HTML

Son herramientas que permiten organizar la informacion de manera tabular, para implementarlas debemos colocar la etiqueta <table en la pagina y dentro de la misma las etiquetas que la conforman.

¿Poseen Estructura Semantica?

A partir de HTML5, a la etiqueta menciona anteriormente se le incoporaron nuevas elementos, que permite identificar con mayor facilidad las partes de la misma. Cabe memcionar que las etiquetas <tr> (Table Row) y <td> (Table Division) se implementan dentro de la estructura, representada por las siguientes etiquetas

  • THEAD Cabecera de Tabla, cuyo contenido son los encabezados que se representan con la etiqueta <th>
  • TBODY Cuerpo de Tabla, que contiene los datos a ser representados. Puede contener celdas de encabezado.
  • TFOOT Pie de Tabla, contiene datos complementarios como totales, maximos, minimos, etc. y es opcional
