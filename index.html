FrontEnd

Desarrollo del lado cliente (Browser), es decir, toda caracteristica o funcionalidad que el navegador puede interpretar o ejecutar sin necesidad de la intervencion de un servidor.
Entre los lenguajes que se manejan del lado del FrontEnd se encuentran los siguientes:

  • HTML
  • CSS
  • JS

HTML

Corresponde a un lenguaje de marcado cuya sigla significa HyperText Markup Language, utiliza las etiquetas para identificar los diferentes elementos que existen en una pagina y la funcion que cumple cada uno dentro de la misma. El orden de los factores es sumamente importante ya que altera el resultado de lo que muestra el navegador.

Las etiquetas tambien poseen atributos, a los que les corresponde un valor definido por el desarrollador. Las mismas se clasifican, de la siguiente manera, segun su disposicion dentro de la pagina:

  • De bloque: Son aquellas que ocupan todo el ancho de contenedor, generan salto de linea y se les puede dar tamaño. Funcionan principalmente como contenedores.
  • De Linea: Son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, no se les puede dar tamaño y funcionan principalmente como contenido.
